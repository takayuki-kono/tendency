# ドキュメント更新ルール

コードに変更を加えた場合、以下のドキュメントも合わせて更新すること:

1. **pipeline_specs.md** - パイプライン仕様書
   - フィルタ項目の追加・削除
   - 処理フローの変更
   - 定数・設定の変更

2. **scripts_overview.md** - スクリプト一覧・概要
   - スクリプトの追加・削除
   - 引数やオプションの変更
   - 入出力の変更

## チェックリスト

- [ ] 新しいフィルタを追加した → 両方のmdに記載
- [ ] 引数を追加/削除した → scripts_overview.md に記載
- [ ] フォルダ構造を変更した → 両方のmdのディレクトリ構造セクションを更新
- [ ] スクリプトを追加した → scripts_overview.md に追加
